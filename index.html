Dragon Project Revision And Reflection

Thoughts on Progressive Enhancement

Progressive Enhancement is a web design ethos that focuses on the content on a page more than anything else. The idea is to build a core and then work out from there. It seems to me that this results in more modular code and provides a more egalitarian browsing experience by ensuring your base elements work flawlessly on, like, Netscape 2.0 or whatever. I thought that this website explained it really well. They emphasized that PE is about more initial planning and thought so that you can cover all your bases and have a website that you can guarantee will be good for everyone. And then from there you can start working on flashy elements.
